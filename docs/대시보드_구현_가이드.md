# 대시보드 구현 가이드

## Flutter 프로젝트 구조 (실무 Feature-First)

```
lib/
├── main.dart
├── shared/
│   └── layout/
│       ├── app_scaffold.dart      # 메인 스캐폴드 (AppBar, endDrawer, body)
│       └── app_menu_drawer.dart  # 햄버거 메뉴 드로어
├── features/
│   ├── dashboard/
│   │   ├── model/
│   │   ├── data/
│   │   └── ui/
│   │       ├── dashboard_screen.dart
│   │       ├── placeholder_screen.dart
│   │       └── widgets/
│   │           ├── summary_card.dart
│   │           ├── traffic_line_chart.dart
│   │           ├── usage_line_chart.dart
│   │           ├── traffic_trend_chart.dart
│   │           ├── donut_chart_widget.dart
│   │           └── section_header.dart
│   └── auth/
│       └── ui/
│           └── login_page.dart
└── docs/
```

## 구조 설명

| 폴더 | 역할 |
|------|------|
| `shared/` | 앱 전역 공통 (레이아웃, 유틸) |
| `shared/layout/` | 메인 스캐폴드, 메뉴 드로어 |
| `features/` | 기능별 모듈 (feature-first) |
| `features/dashboard/` | 대시보드 화면, 차트, KPI |
| `features/auth/` | 로그인/회원가입 |

## 공유 항목 (웹·플러터 동일)

- **화면 구성**: 통합 대시보드, 실시간 모니터링, 알림, QR 조회, 자동 리포트, 로그인
- **메뉴 구조**: 5개 카테고리 + 로그인
- **데이터**: 동일 API, 동일 스키마
